ArcAI — Ask, Reflect, Create

ArcAI is a free multi-modal AI assistant by Win The Night Productions. Chat with every major model, generate images, run real-time voice conversations, write code with a live preview canvas, draft prose, and search the web — all in one app at askarc.chat.

What ArcAI does

  • Unlimited chats with GPT, Claude, Gemini, and Grok models.
  • Real-time voice mode — talk to Arc like a phone call.
  • AI image generation with GPT-Image-2.
  • Code canvas that builds working web apps with a live preview.
  • Writing canvas for long-form prose, briefs, and email.
  • Deep Search — web research with cited AI summaries.
  • Document analysis for PDF, DOCX, and images.
  • Persistent memory and searchable chat history.
  • Shared chats for collaborating with up to 6 people.

Pricing

ArcAI is free forever. The Free plan includes unlimited chats with every model, unlimited voice, Deep Search, shared chats, web publishing, document analysis, memory, canvases, and code generation. Permanent accounts include 20 generated or edited image outputs per UTC day.

There is no paid plan, checkout, or feature paywall. People who want to help can support Win The Night voluntarily. See everything included.

How ArcAI compares

ArcAI is a free alternative to ChatGPT, Claude, Gemini, and Perplexity. Features that those services put behind paid subscriptions — voice mode, image generation, web search, code execution, document uploads — are included in the free tier of ArcAI. The only account-level allowance is 20 image outputs per UTC day.

Frequently asked questions

Is ArcAI really free?
Yes. Every feature is free forever. Permanent accounts include 20 generated or edited image outputs per UTC day.
Do I need an account?
Yes — sign in with Google or email. Accounts let ArcAI save your chats, memories, canvases, and settings across devices.
Which AI models does ArcAI use?
ArcAI routes across GPT, Claude, Gemini, and Grok families depending on the task, plus GPT-Image-2 for images and Perplexity Sonar for web search.
Can ArcAI write and run code?
Yes. The code canvas generates functional web apps with a live preview and includes one-tap publishing to a live URL.
Can ArcAI talk out loud?
Yes. Voice mode is a free, unlimited real-time conversation with Arc using OpenAI's Realtime API.
Where can I get support?
Visit askarc.chat/support. Signed-in users get an in-app ticket portal. Anonymous email support is coming soon.
